#BLIP#

Blip is a 2D multiplayer arcade game inspired by Flappy Bird, developed in Unity using C#. This was my first game development project, where I designed and hand-crafted my own character sprites and implemented both single-player and two-player game modes. The game features smooth movement mechanics, including a custom gliding system, and supports multiplayer gameplay, helping me gain practical experience in game physics, UI design, player controls, and networking concepts.
